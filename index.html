Abstract of the Project
Given a picture captured from a camera hooked up to a vehicle moving on a road during which captured road might not be leveled, or have clearly described edges then road detection from one image will be applied to search out the road in a picture so it might be used as a district in automation of driving system within the vehicles for moving the vehicle in correct road. During this method of finding the road within the image captured by the vehicle, we are able to use some algorithms for vanishing point detection, exploitation of Hough Transformation Space, finding the region of interest, edge detection exploitation, Canny edge detection for road detection. We have a tendency to use thousands of pictures of various roads to coach our model so the model might notice the road as a result within the new image processed through the vehicle,simultaneously it also detects objects on the road using a haar cascade.

Step 2:Applying different-different Algorithms

    1.)HSV COLOR SELECTION : Converting videos to frames and than the Lane lines in the test the images are in white or yellow. We need to choose the most suitable color space, that clearly highlights the lane lines. We applied color selection to the original RGB images, HSV images, and HSL images, and found out that using HSL will be the best color space to use.

    2.GAUSSIAN BLUR: The Canny edge detection algorithm measures the intensity gradients of each pixel. So, we need to convert the images into grayscale in order to detect edges. Apply Gaussian Smoothing : Since all edge detection results are easily affected by image noise, it is essential to filter out the noise to prevent false detection caused by noise. To smooth the image, a Gaussian filter is applied to convolve with the image. It will slightly smooth the image to reduce the effects of obvious noise on the edge detector.

     

    3.) CANNY EDGE DETECTION: We need to detect edges in the images to be able to correctly detect lane lines. The Canny edge detector is an edge detection operator that uses a multi-stage algorithm to detect a wide range of edges in images. The Canny algorithm involves the following steps:

  1. the intensity gradients of the image.
  2. Apply non-maximum suppression to get rid of spurious response to edge detection.
  3. Apply a double threshold to determine potential edges.
  4. Track edge by hysteresis

    5. 4.) REGION OF SELETION : We're interested in the area facing the camera, where the lane lines are found. So, we'll apply region masking to cut out everything else. And we will cut the a polygon shape in the image and than track the lane accordingly.Bounding Boxes is the most important way to resolve this problem.

         

         

       

       

    5.) Hough Transformation : The Hough transform is a technique which can be used to isolate features of a particular shape within an image. Basically we use it to detected the lane lines in the region of selection in the particular area selected.

Haars-Cascade

Applying Major Algorithms Haar-Cascade and Averaging and extrapolating the lane lines

1.) HAARS-CASCADE ALGORITHM :We can load any haar-cascade XML file using cv2.CascadeClassifier function.Once the cascade is loaded in OpenCV, we can call the detector function. It lists coordinates (x, y, w,h) of bounding boxes around the detected object.

   

2.) AVERAGE-SLOPE-INTERCEPT :We have multiple lines detected for each lane line. We need to average all these lines and draw a single line for each lane line. We also need to extrapolate the lane lines to cover the full lane line length.
